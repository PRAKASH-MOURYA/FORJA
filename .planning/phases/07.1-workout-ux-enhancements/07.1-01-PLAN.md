**Change 1: PopScope back interception**

Wrap the existing `Scaffold(...)` return value in a `PopScope` widget:
```dart
return PopScope(
  canPop: false,
  onPopInvokedWithResult: (didPop, result) async {
    if (didPop) return;
    await showModalBottomSheet(
      context: context,
      isScrollControlled: true,
      backgroundColor: Colors.transparent,
      builder: (_) => SessionGuardSheet(
        onResume: () => Navigator.of(context).pop(), // dismiss sheet
        onDiscard: () {
          Navigator.of(context).pop(); // dismiss sheet
          context.pop();               // pop workout screen
        },
      ),
    );
  },
  child: Scaffold(...)   // existing scaffold unchanged
);
```

Also update the AppBar `leading` IconButton so it also triggers the guard:
```dart
leading: IconButton(
```
