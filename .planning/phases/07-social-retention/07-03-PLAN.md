RepaintBoundary capture pattern (from research):
```dart
Future<void> sharePrCard(GlobalKey cardKey, BuildContext context) async {
  final boundary = cardKey.currentContext!.findRenderObject() as RenderRepaintBoundary;
  final image = await boundary.toImage(pixelRatio: 3.0);
  final byteData = await image.toByteData(format: ui.ImageByteFormat.png);
  final bytes = byteData!.buffer.asUint8List();
  final dir = await getTemporaryDirectory();
  final file = File('${dir.path}/pr_card_${DateTime.now().millisecondsSinceEpoch}.png');
  await file.writeAsBytes(bytes);
  // iPad requires sharePositionOrigin:
  Rect? origin;
  if (context.mounted) {
    final box = context.findRenderObject() as RenderBox?;
    origin = box != null ? box.localToGlobal(Offset.zero) & box.size : null;
  }
  await SharePlus.instance.share(ShareParams(
    files: [XFile(file.path)],
    text: 'Just hit a new PR on FORJA!',
    sharePositionOrigin: origin,
  ));
}
```
